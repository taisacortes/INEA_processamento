# INEA_processamento
Scripts em R para limpeza e processamento dos dados de poluição e variáveis meteorológicas do INEA

Etapas
1) Remoção de títulos e cabeçalhos dos arquivos em csv 
2) Identificação e extração dos campos de interesse
3) Formatação dos campos
4) Criação do banco final no formato: data, poluentes e coordenadas
